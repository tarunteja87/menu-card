# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-11
### Added
- Location & Directions admin tab with footer address strip and Google Maps button.
- Embedded Google Maps preview support in the footer.
- Delivery platform buttons support for Swiggy, Zomato, Magicpin, Dunzo, Blinkit, and custom links.
- Dine-in/reservation app buttons support for District, Dineout, EazyDiner, OpenTable, and custom links.
- Configurable link button placement: footer, share bar, or both.
- Browser tab title and favicon customization for the public menu.

## [1.0.0] - 2026-04-11
### Added
- Initial release of the `menu-card` digital restaurant menu.
- Public menu display with hero section, category navigation, menu cards, and footer.
- Admin management panel for adding, editing, and deleting menu items.
- Support for hierarchical parent/subcategory organization.
- Branding controls for restaurant name, tagline, hero image, hours, phone, and address.
- Five visual themes: Ember Gold (dark), Cream & Gold, Cyber Neon, Blush & Rose, and Midnight Blue.
- WhatsApp share button and QR code generation with customizable messaging and styling.
- Feature toggles for announcement banner, table number input, social links, and footer branding.
- JSON import/export of menu configuration.
- Local storage persistence for menu data.
- Optional Firebase Realtime Database support for live synchronization.
